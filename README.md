# Elexira Glow

Premium pharma collagen skincare with only 5 essential ingredients for maximum results.

## Getting Started

### Prerequisites

- Node.js & npm

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion
- Lenis
